# Sistem rapot python

Mini project sistem rapot sederhana menggunakan Python CLI.

## Fitur
- Login Guru/Siswa
- Input data siswa
- Daftar nilai
- Cari siswa
- Update data siswa
- Daftar siswa lulus
- Daftar siswa tidak lulus
- Ranking/Leaderboard
- Rata-rata nilai siswa
- Hapus data siswa
- penyimpanan JSON

## Teknologi
- Python
- JSON
- OOP
- Git & GitHub
